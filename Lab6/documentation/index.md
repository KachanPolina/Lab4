## Інформація про репозиторій 
*** 
Цей репозиторій був створений для навчання. Починаючи з 4 лабораторної роботи тут виконуються різні завдання (лабораторні роботи 1, 2 та 3 є у репозиторії [Studying](https://github.com/KachanPolina/Studying).

В цьому репозиторії є:
- проєкт для роботи з лінтерами та форматерами для мови Python (корень проєкту).
- проєкт для роботи з тестами на мові Python.
-  відображення документації за допомогою GitHub Pages.

Репозиторій [Studying](https://github.com/KachanPolina/Studying) містить:
- основи роботи з Git (корень проєкту).
- встанолвення залежностей проєкту для Python та Node.js.
- розгортання проєкту на С++.
